欢迎使用！

  1. 终于等到您，诚挚您的到来！

  2. 本开发手册就支付+钱麦的技术对接进行详细介绍，通过该指南可以完成支付+的产品技术接入。

  3. 技术对接过程中，所需要的SDK及DEMO，可请前往资源下载。

  4. 如若您需要在线API调试或本机运行，所需要测试资源（AppKey+AppSecret），可联系商户合作。

  5. 如您已经是合作客户，获取AppKey+AppSecret，可以登录到支付+商户后台->"安全管理"->"秘钥管理"

  6. 为方便技术开发小伙伴尽快完成技术对接，请您先了解一些重要政策信息，可以点击用户帮助中心。
