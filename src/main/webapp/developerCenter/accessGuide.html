快速入门

支付+是面向企业提供的支付、账户、账务及营销的综合解决方案产品, 非常抱歉，暂时不对个人提供服务。

开通流程

入网申请

准备入网资料，提交入网申请，具体可联系支付+商务；

联调测试：

对于已确认合作意向的商户，可以同步开启YeePay Plus的技术对接联调。联调测试所需的测试AppKey和AppSecret可以按照上述联系方式，联系支付+商务人员。在调试过程中所遇到任何问题，可以查看帮助中心或按照联系技术支持。

为提升支付+商户标准化、自动化对接，欢迎您反馈在接入过程中遇到的问题或提出意见或反馈，期待您的参与！

产品开通：

提交入网材料审核成功后在1-3个工作日内完成商户所需的产品开通，如超出时间为及时开通，可以进行投诉反馈。

正式商用：

联调测试验证并产品开通成功后，将会获得支付+的正式商编和AppSecret，合作商户的技术小伙伴完成正式商编的替换测试AppKey和AppSecret ，验证没有问题即可登录。3个工作日内完成商户所需的产品开通。

对接准备

如果您已经是支付+的合作客户，对接完成，
请登录商户后台：https://plus.yeepay.com/pp-portal-app/login/to_login；

如果您是未入网的企业商户，可以与支付+商务联系，电话：010-57073793，
电邮：qianmai@yeepay.com；

如果您是准备或正在进行技术对接的商户，请确认入网准备资料，
申请材料准备参见：合作条件。

接入SDK步骤

分别下载 服务端 SDK （链接SDK下载地址），客户端SDK陆续发布中；

安装部署服务端 SDK ，使用服务端 SDK 请求交易对象；
环境建议: 通常JAVA要求 JDK 版本 1.6 及以上。

对接过程，若商户没有正式商编，将采取测试商编和测试Appkey进行验证调试，
若已有正式商编，可以进行替换。
