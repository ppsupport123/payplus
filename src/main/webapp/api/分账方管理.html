1.1分账方管理

1.1.1分账方基本信息注册

【场景介绍】

当企业需要将收款资金分账给渠道方或者其他关联商户时，资金接收方也需要注册，注册信息主要是接收人的基本信息，包括银行卡信息、结算信息等。

【重要说明】分账注册成功后，其账户默认是冻结状态，只有在上传相应资质通过后，才能解冻正常使用。同时，确保分账方的名字务必准确，否则出错会影响后续的分账方结算。

【调用流程】

【请求地址】

环境 接口服务URI
生产环境 /rest/v1.0/merchant/registerLedgerMerchant

1.1.1.1请求参数：

参数名称 参数含义 数据类型 必填(Y\/N) 参数说明
requestNo 商户请求号 String(32) Y
merchantNo 商户编号 String(16) Y
province 分账方所在省份 String（10） Y 分账方所在省,请根据数据 字典「易宝省市编号表.xls」,填 写中文。
city 分账方所在城市 String（20） Y 分账方所在市「易宝省市编号表.xls」,请根据数据字典,填写中文。
customerStyle 商户类型 Enum Y 个人-PERSON
businessLicence 营业执照号 String(30) N customerStyle为ENTERPRISE时必填
signedName 签约名称 String(30) Y customerStyle为PERSON时填姓名，为ENTERPRISE填企业名称。
idCard 身份证号 String(18) N customerStyle为PERSON时必填
legalPerson 法人姓名 String(10) N customerStyle为ENTERPRISE时必填
contactor 联系人 String(20) Y 联系人姓名，最长允许20个汉字
bindMobile 绑定手机 String(11) Y 联系人的手机
email 邮箱 String(30) N
bankAccountNumber 结算银行卡号 String(30) Y 结算用的银行卡号，必须为借记卡
accountName 开户名 String(20) Y 结算用的银行卡号开户名
bankAccountType 卡类型 String(12) Y 个人-PRIVATE_CASH
bankName 开户行 String(50) Y 银行卡所在开户行,请根据数据字典「中国所有银行支行省市库表.xlsx」,填写中文。
bankCardProvince 开户省 String(10) Y 银行卡开户行所在省,请根据数据 字典「易宝省市编号表.xls」,填 写中文。
bankCardCity 开户市 String(20) Y 银行卡开户行所在市「易宝省市编号表.xls」,请根据数据字典,填写中文。
minSettleAmount 最低结算额 Amount Y >= 0.01 的数字,最大18位
riskReserveDay 风险预存期 Int Y 正整数，单位:天例如,当 riskReserveDay=N时, 意味着当天消费的金额,需要在N 个工作日后才会结算出款;如当N=1 时,则当天消费的金额,会在第二个工作日结算出款。

1.1.2分账方资质上传

【场景介绍】

应合规方面的要求，钱麦方需要收集并保存分账方相关资质信息，钱麦方审核资质通过后，分账方才能开展业务。

【请求地址】

环境 接口服务URI
生产环境 /rest/v1.0/merchant/uploadQualifications

1.1.2.1请求参数

参数名称 参数含义 数据类型 必填 参数说明
merchantNo 商户编号 String(32) Y
ledgerNo 分账方编号 String(16) Y
qualificationType 证件类型 Enum (20) Y 身份证正面-ID_CARD_FRONT 身份证背面-ID_CARD_BACK 银行卡正面-BANK_CARD_FRONT 银行卡背面-BANK_CARD_BACK 个人相片-PERSON_PHOTO 营业执照-BUSSINESS_LICENSE 工商证-BUSSINESS_CERTIFICATES 银行开户许可证-BANK_ACCOUNT_LICENCE 组织机构代码证-ORGANIZATION_CODE
imgBase64Buffer 文件流 String(512000) Y 图片转base64编码的字符串
【重要说明】分账方可以分为个人和企业两种类型情况，需要开发者务必仔细确认。

1、关于证件类型，个人类型账户必上传资质:身份证正、反面；个人相片（手持身份证照片）,银行卡正面;

2、企业类型账户必上传资质:企业五证(营业执照,法人身份证正、反面,银行开户许可证,组织机构代码证, 税务登记证)。

3、上关于文件流，上传图片类型支持 jpg、jpeg、png,文件大小限制 512KB。

1.1.2.2返回参数

参数名称 参数定义 参数说明
code 返回码 请参考附录：返回码列表
message 返回消息 返回码的详细说明
