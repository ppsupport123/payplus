3.2 商户现金红包

3.2商户现金红包

3.2.1创建现金红包

【场景介绍】

创建一次营销活动所需的现金红包，用户领取红包后，红包直接入到用户资金账户中，用户可提现到银行卡。

如果是平台模式，发现金红包方目前仅限平台。现金红包活动创建前商户需保证资金账户中资金充足。

请求地址：

环境 接口服务URI
生产环境 \/rest\/v1.0\/merchant\/createRedPacketDefine

3.2.1.1请求参数

参数名称 参数含义 数据类型 必填 参数说明
requestNo 请求流水号 String(32) Y
sendMerchantNo 创券方商户编号 String(16) Y 目前只有平台商
name 活动名称 String(32) Y
value 面值 Amount Y 单位：元限制大于等于0.01，最大18位
totalCount 数量 Int Y 限制大于等于1的正整数

3.2.1.2返回参数

参数名称 参数定义 参数说明
code 返回码 请参考附录：返回码列表
message 返回消息 返回码的详细说明

3.2.2发送现金红包

【场景介绍】

将现金红包发送给商户自己的用户，建立用户与现金红包起对应关系。

请求地址：

环境 接口服务URI
生产环境 \/rest\/v1.0\/merchant\/sendRedPacket

3.2.2.1请求参数:

参数名称 参数含义 数据类型 必填 参数说明
defineRequestNo 红包定义编号 String(32) Y
merchantNo 商户编号 String(16) Y 用户所属商户的商编
merchantUserId 领券方商户用户标识 String(32) Y
expireTime 过期时间 Int(1,44640) Y 红包发送后未领取自动撤回的过期时间，单位（分钟）限制1<=Time<=44640(31天)

3.2.2.2返回参数:

参数名称 参数定义 参数说明
couponNo 现金红包编号
code 返回码 请参考附录：返回码列表
message 返回消息 返回码的详细说明

3.2.3领取现金红包

【场景介绍】

用户领取现金红包时，需要调用此接口，实名用户的账户余额会增加。

请求地址：

环境 接口服务URI
生产环境 \/rest\/v1.0\/user\/receiveRedPacket

3.2.3.1请求参数:

参数名称 参数含义 数据类型 必填 参数说明
couponNo 卡券编号 String(32) Y
merchantNo 商户编号 String(16) Y 用户所属商户的商编
merchantUserId 领券方商户用户标识 String(32) Y

3.2.3.2返回参数:

参数名称 参数定义 参数说明
code 返回码 请参考附录：返回码列表
message 返回消息 返回码的详细说明

3.2.4用户未领取红包查询

【场景介绍】

查询某个用户所有未领取红包。

备注：如传入创建的红包定义编号，则查询的是该红包未领取信息。

请求地址：

环境 接口服务URI
生产环境 \/rest\/v1.0\/user\/queryUnReceivedRedPackets

3.2.4.1请求参数:

参数名称 参数含义 数据类型 必填 参数说明
defineRequestNo 红包定义编号 String(32) N
merchantNo 商户编号 String(16) Y 用户所属商户的商编
merchantUserId 领券方商户用户标识 String(32) Y

3.2.4.2返回参数:

参数名称 参数定义 参数说明
couponList 红包列表 红包列表说明详见下文注解
code 返回码 请参考附录：返回码列表
message 返回消息 返回码的详细说明

红包列表说明注解：参考3.1.4的注释说明，但其卡券类型为现金红包，即couponType：CASH_RED_PACKET。

3.2.5红包定义信息查询

【场景介绍】

查询已创建现金红包目前的整体情况，包括总金额、数量、面值、状态等。

请求地址：

环境 接口服务URI
生产环境 \/rest\/v1.0\/merchant\/queryRedPacketDefine

3.2.5.1请求参数:

参数名称 参数含义 数据类型 必填 参数说明
defineRequestNo 红包定义编号 String(32) Y
sendMerchantNo 发券方商户编号 String(16) Y

3.2.5.2返回参数:

参数名称 参数定义 参数说明
name 名称
value 面值
totalCount 总数量
totalAmount 总金额
unSendCount 未发送数量
unreceivedCount 未领取数量
status 状态 初始化-INIT
refundAmount 退款金额 单位：元，目前返回为空
freezeAmount 冻结资金金额 单位：元，目前返回为空
code 返回码 请参考附录：返回码列表
message 返回消息 返回码的详细说明
