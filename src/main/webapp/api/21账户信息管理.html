2.1账户信息管理

2.1.1用户账户注册（会员注册）

【场景介绍】

当有用户资金账户的场景时，比如用于充、转、提功能，需要调用本接口创建用户，商户用户标识是用户在商户平台的唯一标识，一个用户标识+身份证号只允许注册一次。

请求地址：

环境 接口服务URI
生产环境 \/rest\/v1.0\/user\/register

2.1.1.1请求参数

参数名称 参数含义 数据类型 必填 参数说明
requestNo 商户请求号 String(32) Y
merchantNo 商户编号 String(16) Y
merchantUserId 商户用户标识 String(32) Y

2.1.1.2返回参数

参数名称 参数定义 参数说明
memberNo 会员编号 商户用户在钱脉的会员编号
code 返回码 请参考附录：返回码列表
message 返回消息 返回码的详细说明

2.1.2发起实名认证

【场景介绍】

根据相关法规要求，用户资金账户的充值、提现、转账、账户余额支付、绑卡支付时，需要实名认证后才可以使用，实名认证时跳转到钱脉H5页面，用户按照页面提示自行操作完成。

备注：目前用户的实名认证是通过对其银行卡信息的认证完成的，用户实名认证通过之后，会引导用户设置支付密码，以保证支付账户的安全性，该操作和实名操作都是在钱脉的H5页面完成，设置完成之后，结果同步\/异步反馈给商户，且在用户操作完成之后，返回商户端页面。

【页面展示】

【调用流程】

请求地址：

环境 接口服务URI
生产环境 \/rest\/v1.0\/user\/auth

2.1.2.1请求参数

参数名称 参数含义 数据类型 必填 参数说明
requestNo 商户请求号 String(32) Y
merchantNo 商户编号 String(16) Y
merchantUserId 商户用户标识 String(32) Y
serverCallbackUrl 服务器通知地址 String（512） Y
webCallbackUrl 页面回调地址 String(512) Y 实名完成后返回商户的地址
returnUrl 返回地址 String(512) Y 钱脉第一个H5页面左上角点“返回”的回退地址

2.1.2.2返回参数

参数名称 参数定义 参数说明
redirectUrl 重定向地址 跳转到钱脉实名页面的地址
code 返回码 请参考附录：返回码列表
message 返回消息 返回码的详细说明

2.4.1.3主动通知：

通知方式：将参数通知到请求时传入的serverCallbackUrl。

通知机制：仅异步通知一次，如通知失败，则每3分钟补偿通知1次，共通知3次。

通知参数：同4.1.2.2订单查询的返回参数。

2.1.3用户证件上传

【场景介绍】

  根据相关法规要求，第三方支付公司对用户身份信息验证的内容及渠道，与账户的累计支付限额有相关性，故用户上传较多的身份信息，第三方支付公司便可通过较多渠道对用户身份进行认证，其账户的等级和权限就会较高。第三方支付公司留存用户的身份证影印件也是法规要求，故尽量引导用户上传。

备注：用户的该操作需要在钱脉页面完成，故需要商户端向钱脉发起用户资质上传的

请求，钱脉将结果同步\/异步反馈给商户，且在用户操作完成之后，返回商户端页面。

请求地址：

环境 接口服务URI
生产环境 \/rest\/v1.0\/user\/authCredentials

2.1.3.1请求参数

参数名称 参数含义 数据类型 必填 参数说明
requestNo 商户请求号 String(32) Y
merchantNo 商户编号 String(16) Y
merchantUserId 商户用户标识 String(32) Y
fileType 文件类型 Enum(12) Y 文件类型（正面和反面需要分别调接口）：身份证正面-ID_CARD_FACE身份证反面-ID_CARD_BACK
file 文件 Srring(512000) Y

2.1.3.2返回参数

参数名称 参数定义 参数说明
code 返回码 请参考附录：返回码列表
message 返回消息 返回码的详细说明

2.1.4用户信息查询

【场景介绍】

方便商户查询用户的基本信息，主要包括基本信息、状态、认证情况等。

请求地址：

环境 接口服务URI
生产环境 \/rest\/v1.0\/user\/queryUserInfo

2.1.4.1请求参数

参数名称 参数含义 数据类型 必填 参数说明
merchantNo 商户编号 String(16) Y
merchantUserId 商户用户标识 String(32) Y

2.1.4.2返回参数

参数名称 参数定义 参数说明
name 商户用户姓名 未实名不返回
registerTime 商户用户注册时间
status 商户用户状态 正常-AVAILABLY
caStatus 商户用户实名状态 未认证-UNVERIFIED
caTime 认证时间 未实名不返回
credentialsType 证件类型 身份证-IDENTITY_CARD
credentialsNo 证件号码 未实名不返回
memberCategory 会员等级 未实名-COMMON_CATEGORY
uploadCertificateAuthStatus 证件上传认证状态 上传证件认证通过-VERIFIED
code 返回码 请参考附录：返回码列表
message 返回消息 返回码的详细说明
